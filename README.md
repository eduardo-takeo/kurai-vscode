## Installation via VS Code

1. Open **Extensions** sidebar panel in VS Code. `View â†’ Extensions`
2. Search for `Kurai`
3. Click **Install** to install it
4. Click **Reload** to reload the editor
5. Code > Preferences > Color Theme > **Kurai**
